Test Environment Selftest Suite
===============================

Selftest test suite is aimed to check implementation of Test Environment API
function and along with this is an example of using Test API.

This test suite runs every day here over
`this Jenkins job <https://oktetlabs.ru/prj/ol/jenkins/job/TE/job/te-selftests/>`_

Run selftest
~~~~~~~~~~~~

For start you need:

.. code::

    hg clone https://oktetlabs.ru/hg/oktetlabs/test-environment
    cd test-environment/suites/selftest
    ./run.sh
