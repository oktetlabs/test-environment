ANNOTATION
The short document should contain all information necessary to understand
and support the implementation of the storage library based on CVS and file
system. 
Particularly following issues should be clarified for file system-based 
implementation:
    * possibility of archieving;
    * remote file systems;
    * configuration string format and processing;
    * file sharing conventions.
    
and for CVS-based implementation:
    * configuration string format and processing;
    * processing of errors returned by CVS.
    
Location of temporary files and portability to MS systems should be described
as well.
