Proteos: primary log format specification

ANNOTATION
The document should provide full description of all messages which may
be stored in raw log. Particularly following elements of raw log should
not be fogotten:
* timestamps;
* owner (subsystem, Test Package, user of the Test Agent);
* string messages storing;
* storing of file content;
* storing of memory dumps.

Specification should allow implementor of Report Generator to start
his work without any additional information.
